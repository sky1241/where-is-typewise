## Editing notes

- Trim any pause longer than 0.8 seconds
- If a tool call takes more than 4 seconds, speed up to 1.5×
- Don't add background music
- Don't add captions unless you mispronounce something — captions waste viewer attention

## What this Loom proves to a reviewer

1. The MCP exists, the tools run, the buyer prompt yields the right output
2. The dashboard exists, has real-looking data, the DACH moat is visible
3. The candidate has a concrete month-1 plan, not vibes
4. The candidate ships in 90 seconds what other applicants stretch into 8-minute decks
